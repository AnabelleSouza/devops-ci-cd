# Pipeline CI/CD Frontend

Projeto desenvolvido para demonstrar a criação de um pipeline de build e deploy.

## Tecnologias
- React
- GitHub Actions
- GitHub Pages

## Pipeline
1. Checkout do código
2. Instalação de dependências
3. Build da aplicação
4. Deploy automático
